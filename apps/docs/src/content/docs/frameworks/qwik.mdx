:::caution[Experimental Qwik 2 support]
Import this adapter from `@vp-tw/likftc/qwik`. It requires the exact `@qwik.dev/core@2.0.0-beta.36` peer and the Qwik Vite optimizer. Qwik 2 is still in beta, so do not float the version range. Its current SVG JSX declarations require an adapter-scoped `skipLibCheck`, and its runtime ESM cannot be evaluated directly by Node without optimizer globals. Likftc source remains fully strict under TypeScript 6 and TypeScript 7; direct Node imports, SSR, and hydration are not supported. This subpath may change in a Likftc minor release until the experimental label is removed.
:::
