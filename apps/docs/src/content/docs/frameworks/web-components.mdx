The package has no Lit dependency. In Lit, use the `repeat()` directive with `entry.key`; in a native Custom Element, use the same key to retain or create owned DOM nodes.

If reconciliation throws, the controller keeps the previous committed state. This prevents a duplicate-ID frame from corrupting the next update.
