TanStack Store owns the current frame. Its official React `useSelector()` hook updates the component,
and Likftc gives the renderer a distinct key when an ID returns before its previous presence exits.

:::caution[TanStack Store pre-1.0 API]
This demo targets `@tanstack/store` 0.11.0. TanStack Store remains pre-1.0, so verify
the bridge after upgrading it.
:::
